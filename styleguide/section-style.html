Colors

Always try to stick to the Primary Colors. Only use Secondary Colors for slight accents or highlights. Secondary Colors should only be dominant when used in a child/sub theme.

Primary Colors

Try to use these colors for everything. Only use these colors where deemed appropriate via this styleguide.

Icons

Users tend to skim through context, looking for subtle clues and markers to determine the intent of a link or block of text. Icons are a great way to make "skimmable" websites. However, to ensure "skimmability", each icon must be used in a unique context that defines what that icon means on the website.

Each icon should only represent one thing or idea, otherwise the user may feel disoriented or frustrated when they get an unexpected outcome.

The Grid

The grid is essential for a website to be responsive - adapting properly for all devices, from phone to desktop to jumbotron! The grid can support a maximum 12 columns of equal size. But don't worry, you can any number of columns, as long as they don't go over 12!

Grid creation

To create a grid you must define rows and then put columns inside those rows.

Grid options

You define column size by using one of 4 dynamic css classes. Each class is designed to adjust column sizes relative to different device sizes

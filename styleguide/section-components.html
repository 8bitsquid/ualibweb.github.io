Scrollfix

The .ui-scrollfix class allows elements to become fixed to the page before a user scrolls past them. This is great for sub-navigation or column headers for large tables of data.

Elements with .ui-scrollfix class will become fixed when they are 30px from the top of the page.

The Pages & Submenus described in the Layout section have scroll fixed menus applied to the by default.

                               <div class="ui-scrollfix">I'm not going to be sticky in the docs! They'd just be annoying.</div>

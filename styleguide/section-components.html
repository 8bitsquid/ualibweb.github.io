Pages & Submenus

Submenus are automatically generated by useing the Page component. This allows content creators to focus on content without having to worry about navigation. Use the <page> tag to start a "menued" page, then add <page-section> tags for each section.

Menu items are assigned using the <h2> heading for each page section. If you want the menu text to be different from a page section's heading, use the title attribute on the <page-section> element.
